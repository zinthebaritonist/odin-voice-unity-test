# CLAUDE.md - 開発方針と制約

## 🚫 絶対禁止事項

### モック・シミュレーション禁止
- **モックは２度と作りません**
- **テストモード、シミュレーションモード等の偽機能は一切実装しません**
- **実際の機能のみを実装します**

### 誠実性の維持
- **嘘はつきません**
- **実装していない機能を「動いている」と言いません**
- **問題がある場合は正直に報告します**

## ✅ 開発原則

1. **実機能のみ実装** - モック、ダミー、シミュレーション等は一切作成しない
2. **正直な報告** - 問題や制限事項を隠さず報告する
3. **実用性重視** - 実際に動作する機能のみを提供する
